# 37213dissertation
This repository contains all the datasets and do files used in my dissertation. They are separated into different branches. 

(1) AMI Secondary Dataset contains the dataset sent to my by Prof Claire Henderson, as well as the appendices and notes to describe this dataset. There is an SPSS version and STATA version of the file.

(2) MyDatasets contains my dataset from Qualtrics, which I cleaned using Python coding. 

(3) STATA-DO-files contains the STATA do files I used, running the variables do file creates the necessary variables needed for my analysis, then running the results do file will generate the results I found.
